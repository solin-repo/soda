Class soda_error

Description
Description | Vars (details) | Methods (details)

Class soda_error stores error messages meant to be displayed for end users.

A typical use case is validation. If a model is not valid, the error messages is added to soda_error::$validation errors. You can display the invalid form and use the error messages to show the user which fields are "in error".
